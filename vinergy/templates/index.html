NAME
    Vinergy: CLI Pastebin within VimEnergy

SYNOPSIS
    <command> | curl -F 'vimcn=<-' $url

DESCRIPTION
    add ?<lang> or /<lang> to resulting url for line numbers and syntax
    highlighting

EXAMPLES
    ~ % cat bin/code.js | curl -F 'vimcn=<-' $url
       ${url}/cbc

    ~ % curl ${url}/cbc?js
    ~ % wget -qO- ${url}/cbc/js
    ~ % firefox ${url}/cbc?js

URLS
    http://img.vim-cn.com/
        A CLI image pastebin
